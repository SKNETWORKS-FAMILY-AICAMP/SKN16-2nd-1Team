# 흡연율 예측 프로젝트

## 1. 프로젝트 개요

### 1.1 프로젝트 소개  
2020–2022년 지방자치단체별 인구·보건·경제 지표를 활용하여, 2023년 흡연율을 머신러닝으로 예측하는 프로젝트입니다.

### 1.2 주제선정 배경  
- **공중보건 강화**: 자치단체별 흡연율 변화를 사전에 예측해 금연 정책 효과를 극대화  
- **데이터 활용**: 행정구역 단위 상세 데이터를 모델에 적용해 지역 특성을 반영

### 1.3 프로젝트 목표  
1. 2020–2022년 데이터를 기반으로 2023년 흡연율 예측  
2. 10종 회귀 모델(RF, XGBoost, LGBM, Stacking 등) 성능 비교  
3. Streamlit 웹 앱으로 대시보드 배포

---

## 2. 데이터분석 및 전처리

### 2.1 데이터셋  
- 출처: 공공데이터포털 “시군구별 흡연율”  
- 기간: 2020–2023년, 전국 350여 개 시군구  
- 주요 컬럼:

## 3. 시연 페이지  
Streamlit 웹 앱을 통해 다음 기능 제공:  
1. **데이터 확인**: 원본 데이터·통계 미리보기  
2. **연도별 트렌드**: 시도/시군구별 지표 시계열  
3. **상관분석**: 변수 간 산점도·상관계수 시각화  
4. **예측 및 튜닝**: 10개 모델 선택·튜닝, 2023년 흡연율 실시간 예측
